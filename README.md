# asteain_keyboard
asteainの作成した、諸人工言語を統一的に扱うことのできる単一のフォントファイルと、入力を支援するソフトウェアキーボード。
現在はアソーグ文字のソフトウェアキーボードのみ実装済み。

オンラインで利用するためのリンク：https://asteain-ninia.github.io/asteain_keyboard/
